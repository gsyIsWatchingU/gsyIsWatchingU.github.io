1. WebSocket

WebSocket 是一种在单个 TCP 连接上进行全双工通信的协议，适用于实时数据传输场景。

特点：全双工通信、低延迟、持久连接

2. WebWorker

WebWorker 是在后台运行的 JavaScript 线程，不会阻塞主线程的 UI 渲染。

特点：多线程、不阻塞 UI、并行计算

3. ServiceWorker

ServiceWorker 是浏览器与网络之间的代理服务器，用于实现离线缓存、推送通知等高级功能。

特点：离线缓存、拦截网络请求、后台同步

4. IndexedDB

IndexedDB 是浏览器内置的 NoSQL 数据库，用于在客户端存储大量结构化数据。

特点：大容量存储、支持索引、异步操作

5. WebGL / OffscreenCanvas

WebGL 是基于 OpenGL ES 的 JavaScript API，用于在浏览器中渲染高性能 2D/3D 图形。

OffscreenCanvas 允许在 Worker 线程中渲染图形，实现离屏渲染。

特点：GPU 加速、硬件加速渲染、游戏/可视化
